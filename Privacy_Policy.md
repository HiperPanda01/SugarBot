### 4. Data Retention
The bot does not collect data permanently **EXCEPT**: 
- Server IDs
- Channel IDs
- Custom reply deletion notice messages
- Banned reaction emojis

The above data are necessary solely to associate bot functionality with the appropriate server configuration. No other data are collected for any other purpose whatsoever.

The data the bot retrieves via the Discord API are processed only as far as necessary for basic functionality for as long as the code function executes. Any long-term logging occurs only within channels of the Discord servers where the bot is used and is fully controlled by those server administrators.

Configuration data are deleted when the bot is removed from a server or when manually cleared by server administrators.
